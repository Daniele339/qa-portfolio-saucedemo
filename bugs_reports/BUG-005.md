ID: BUG-005

Título:
[Functional] Sorting feature displays error message instead of ordering products

Tipo:
Funcional / UI / Interação

Severidade: Alta  
Prioridade: Alta  

Ambiente:
- Browser: Google Chrome 147
- Sistema Operacional: Windows 10
- URL: https://www.saucedemo.com/

Pré-condição:
Usuário autenticado com "error_user"

Passos para reprodução:
1. Acessar a aplicação
2. Realizar login com usuário válido (error_user)
3. Na página inventory.html, localizar o campo de ordenação (dropdown)
4. Selecionar qualquer opção de ordenação (ex: Name A-Z, Price Low-High)

Resultado esperado:
Os produtos devem ser ordenados corretamente de acordo com a opção selecionada

Resultado atual:
Ao selecionar uma opção de ordenação, o sistema exibe a mensagem:
"Sorting is broken! This error has been reported to Backtrace."

Impacto:
O usuário não consegue organizar os produtos, prejudicando a navegação e experiência de uso.
Além disso, a exibição de mensagem técnica compromete a credibilidade da aplicação

Evidência:
(Adicionar screenshot ou vídeo)

Observação:
Possível falha na função de ordenação ou erro de tratamento de exceção no front-end.
Mensagem indica erro interno exposto ao usuário final

Sugestão:
Implementar corretamente a lógica de ordenação e tratar erros de forma amigável ao usuário

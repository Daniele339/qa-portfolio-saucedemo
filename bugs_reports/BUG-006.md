ID: BUG-006

Título:
[Form] Last Name field is not editable on checkout information page

Tipo:
Funcional / UI / Formulário

Severidade: Alta  
Prioridade: Alta  

Ambiente:
- Browser: Google Chrome 147
- Sistema Operacional: Windows 10
- URL: https://www.saucedemo.com/

Pré-condição:
Usuário autenticado com "error_user"

Passos para reprodução:
1. Acessar a aplicação
2. Realizar login com usuário válido (error_user)
3. Adicionar um produto ao carrinho
4. Acessar o carrinho (cart.html)
5. Clicar em "Checkout"
6. Na tela de informações do usuário, localizar o campo "Last Name"
7. Tentar inserir dados no campo

Resultado esperado:
O campo "Last Name" deve permitir a inserção de texto normalmente

Resultado atual:
O campo "Last Name" não permite digitação, impedindo o preenchimento do formulário

Impacto:
O usuário não consegue completar o formulário de checkout, bloqueando a finalização da compra

Evidência:
(Adicionar screenshot ou vídeo)

Observação:
Possível problema de atributo HTML (readonly/disabled) ou falha no tratamento de input no front-end

Sugestão:
Verificar propriedades do campo e garantir que esteja habilitado para edição durante o preenchimento do formulário

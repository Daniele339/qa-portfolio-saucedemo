ID: BUG-009

Título:
[Navigation] Inventory page loads blank after returning from cart

Tipo:
Funcional / Navegabilidade / Possível Intermitente

Severidade: Alta  
Prioridade: Alta  

Ambiente:
- Browser: Google Chrome 147
- Sistema Operacional: Windows 10
- URL: https://www.saucedemo.com/

Pré-condição:
Usuário autenticado com "standard_user"

Passos para reprodução:
1. Acessar a aplicação
2. Realizar login com usuário válido (standard_user)
3. Na página inventory.html, clicar no ícone do carrinho
4. Na página cart.html, clicar no botão "Back" ou "Continue Shopping"
5. Observar a página inventory.html

Resultado esperado:
A página inventory.html deve ser exibida com a lista de produtos carregada corretamente

Resultado atual:
A página inventory.html é exibida em branco, sem carregar os produtos

Impacto:
O usuário fica impedido de continuar navegando ou selecionando produtos, interrompendo o fluxo de compra

Evidência:
(Adicionar screenshot ou vídeo)

Observação:
Possível falha no carregamento de estado da aplicação ou renderização após navegação.
Comportamento pode estar relacionado a cache, sessão ou problema no front-end (JavaScript)

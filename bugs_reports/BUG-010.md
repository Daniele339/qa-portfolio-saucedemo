ID: BUG-010

Título:
[Security] Unauthorized access to protected pages without authentication

Tipo:
Segurança / Funcional

Severidade: Alta  
Prioridade: Alta  

Ambiente:
- Browser: Google Chrome 147
- Sistema Operacional: Windows 10
- URL: https://www.saucedemo.com/

Pré-condição:
Usuário não autenticado na aplicação

Passos para reprodução:
1. Abrir o navegador
2. Sem realizar login, acessar diretamente as URLs:
   - https://www.saucedemo.com/inventory.html
   - https://www.saucedemo.com/cart.html

Resultado esperado:
O sistema deve redirecionar o usuário para a página de login ao tentar acessar páginas restritas sem autenticação

Resultado atual:
O sistema permite acesso direto às páginas protegidas sem exigir autenticação

Impacto:
Usuários não autenticados podem acessar áreas restritas do sistema, comprometendo a segurança e integridade da aplicação

(Adicionar screenshot ou vídeo)

Observação:
Indica ausência de controle de sessão ou validação de autenticação no lado do cliente/servidor

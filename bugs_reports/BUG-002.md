ID: BUG-002

Título:
[Auth] User is redirected to login when accessing cart after successful authentication

Tipo:
Funcional / Autenticação / Navegação

Severidade: Alta  
Prioridade: Alta  

Ambiente:
- Browser: Google Chrome 147
- Sistema Operacional: Windows 10
- URL: https://www.saucedemo.com/

Pré-condição:
Usuário autenticado com "problem_user"

Passos para reprodução:
1. Acessar a aplicação
2. Realizar login com usuário válido (problem_user)
3. Na página inventory.html, clicar no ícone do carrinho

Resultado esperado:
O sistema deve redirecionar o usuário para a página cart.html, exibindo os itens do carrinho

Resultado atual:
O sistema redireciona o usuário para a página de login com a mensagem:
"Epic sadface: You can only access '/cart.html' when you are logged in."

Impacto:
O usuário não consegue acessar o carrinho mesmo após autenticação, impedindo a continuidade do fluxo de compra

Evidência:
(Adicionar screenshot ou vídeo)

Observação:
Possível falha no gerenciamento de sessão ou autenticação do usuário.
O sistema pode não estar mantendo o estado de login corretamente após autenticação

ID: BUG-007

Título:
[Logic] Cart contains item before user interaction after login

Tipo:
Funcional / Lógica / Estado

Severidade: Alta  
Prioridade: Alta  

Ambiente:
- Browser: Google Chrome 147
- Sistema Operacional: Windows 10
- URL: https://www.saucedemo.com/

Pré-condição:
Usuário não possui itens adicionados ao carrinho antes do login

Passos para reprodução:
1. Acessar a aplicação
2. Realizar login com usuário válido (standard_user)
3. Na página inventory.html, observar o ícone do carrinho

Resultado esperado:
O carrinho deve estar vazio ao iniciar a sessão, sem itens previamente adicionados

Resultado atual:
O carrinho já contém um item sem que o usuário tenha realizado qualquer ação

Impacto:
Pode causar inconsistência no processo de compra, levando o usuário a finalizar pedidos com itens não desejados

Evidência:
(Adicionar screenshot ou vídeo)

Observação:
Possível falha no gerenciamento de estado ou persistência de sessão (cache/local storage)
